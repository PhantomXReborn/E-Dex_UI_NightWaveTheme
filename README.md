# Russian Violet Theme for EDEX UI

A deep, moody, and immersive theme for [EDEX UI](https://github.com/GitSquared/edex-ui), blending rich purples, muted violets, and soft lavenders on near-black backgrounds. Designed for coders and terminal enthusiasts who love a futuristic, elegant aesthetic.
